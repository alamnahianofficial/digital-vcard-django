# Digital VCard (Django)

This is a simple digital business card I built using Django.

The idea was to create a clean personal page where someone can:
- see my basic info
- access my social links
- download my contact directly as a `.vcf` file

I also added a small custom admin panel so I can update everything without touching code.

---

## Live Project

https://Nahian.pythonanywhere.com

## What it does

- Shows a personal profile (name, bio, skills, etc.)
- Lets visitors download my contact
- Includes social/media links
- Has a private admin page to edit everything
- Supports profile photo upload
